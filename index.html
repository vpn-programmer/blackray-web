Нужно ли что-то устанавливать на сайт?

Нет. Это статичный лендинг, который открывается как обычный сайт. Для GitHub Pages нужен только репозиторий с index.html.

Можно ли поменять тексты и кнопки позже?

Да. Меняешь пару строк в HTML, коммитишь — и GitHub Pages автоматически публикует новую версию.

Можно ли подключить свой домен?

Да. Когда купишь домен, его можно привязать к GitHub Pages без смены самой структуры сайта.
